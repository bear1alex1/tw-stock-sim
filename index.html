📋 Firebase 設定說明
  1. 前往 Firebase Console 建立專案
  2. Authentication → Sign-in method → 啟用 Google
  3. Firestore Database → 建立資料庫 → 套用安全規則
  4. 專案設定 → 新增 Web 應用程式 → 複製 Config 物件
  5. 將 Config 填入 app.js 頂部的 FIREBASE_CONFIG
  6. Authentication → 已授權網域 → 加入你的網域（或 localhost）
Firestore 安全規則： 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /users/{uid} {
      allow read, write: if request.auth != null
                         && request.auth.uid == uid;
    }
    match /{document=**} {
      allow read, write: if false;
    }
  }
}
